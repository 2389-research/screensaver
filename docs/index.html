macOS 14+ · Native Swift · Open Source

Code Agent Screensaver

Your Mac, pretending to be a room full of AI engineers. A native screensaver that simulates a giant tmux grid of Code Agent instances all hacking away.

Download .saver View on GitHub

Features

▦

Real tmux Chrome

Text-character borders with proper junction characters (┼ ├ ┤ ┬ ┴), status bar with session name, window list, and live clock.

❯

Code Agent TUI

Faithful reproduction of the Code Agent interface: ❯ prompt, thinking spinner, tool call blocks with rounded corners, streaming responses.

▒

6 Dark Themes

Each pane gets a different color scheme — Catppuccin Mocha, Tokyo Night, Dracula, Nord, Gruvbox, Solarized. No two panes look the same.

⏰

Realistic Timing

Delta-time animation at human-like speeds. Prompts type at 40 chars/sec, responses at 60, tool calls at 200. Thinking spinners cycle at 100ms.

▦

Living Layout

Panes slowly split, close, and resize over time — like a developer managing their tmux session. Staggered startup sequence brings panes to life one by one.

⚡

21 Bundled Sessions

From building REST APIs to debugging memory leaks to the infamous "everything goes wrong" deploy disaster. Realistic tool usage patterns throughout.

Color Themes

Each pane randomly selects from six carefully chosen dark palettes.

❯ build api Catppuccin
❯ fix bug Tokyo Night
❯ add auth Dracula
❯ deploy Nord
❯ test Gruvbox
❯ search Solarized

Install

1

Download

Grab the latest .saver.zip from GitHub Releases

2

Unzip & Move

Move .saver to ~/Library/Screen Savers/

3

Activate

System Settings → Screen Saver → select it

# Or build from source:
make install
