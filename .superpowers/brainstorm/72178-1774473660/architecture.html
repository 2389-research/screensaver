Architecture Overview

ClaudeCodeScreenSaver.saver — Core Animation based

Component Architecture
ScreenSaverView (entry point)
Subclass of ScreenSaverView. Owns the root CALayer tree.
Handles animateOneFrame() tick, preferences sheet.
|
TmuxLayoutEngine
Manages pane tree (splits, resizes, adds, removes).
Outputs frame rects for each pane. Handles slow evolution.
|
PaneView (x N)
CALayer container per pane.
Contains: TerminalRenderer + TmuxChrome
TmuxChromeLayer
Green status bar, pane borders,
window tabs, pane numbers.
|
TerminalRenderer
CATextLayer grid. Renders attributed strings
with ANSI colors. Handles cursor, scrolling.
SessionPlayer
Reads JSONL corpus. Drives typing speed,
pauses, tool blocks. One per pane.
|
JSONL Session Corpus (bundled resource)
~20-30 hand-crafted sessions. Each is a sequence of events:
prompt, response_chunk, tool_call, tool_result, thinking, etc.
